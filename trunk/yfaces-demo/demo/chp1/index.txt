
{title} 
Basics

{usecase}
Pass property values to component from view (push/inject values)


{parts}
- generische ermittlung der xhtml views: getResources(url minus host)
- generische ermittlung der component klassen (new ycomponentinfo)

{long}
The easiest component just acts as basic data container which displays values.
Values may be calculated by the component or pushed/injected/passed from the view.<br/>
<br/>
The component is used multiple times at this page via <yf:showProductV1/>;<br/>
Each component view is backed by an own component instance (somthing which is hard to achieve with plain JSF ManagedBeans)
Each setter of the component class is available as injectable property.<br/>
Property values can be injected as tag attributes <yf:showProductV1 code="value" />;
   	
{example-ex1}
<yf:demo1/>

{example-ex2}
<yf:demo1 code="FRUIT-1"/>
